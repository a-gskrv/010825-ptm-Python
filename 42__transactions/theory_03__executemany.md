### Метод `executemany` —

позволяет выполнить многократную (пакетную) операцию записи данных из двумерного массива  
(или другой итерируемой коллекции) без явного цикла на стороне пользователя.

Предыдущий пример:

```python

        rows = [
            ("Keyboard", 2, 45.50, "2024-06-15"),
            ("Mouse", 5, 20.00, "2024-06-16"),
            ("Monitor", 1, 150.00, "2024-06-17"),
            ("USB Cable", 10, 5.50, "2024-06-18"),
            ("Webcam", 3, 70.00, "2024-06-19"),
        ]

        cursor.executemany(
            "INSERT INTO sales (item_name, quantity, price, sale_date) VALUES (%s, %s, %s, %s)",
            rows
        )
```

⚠️ Строго говоря, цикл здесь всё равно присутствует.  
Но не в коде пользователя, а внутри пакета `pymysql`
